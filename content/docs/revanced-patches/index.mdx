**Hyro's ReVanced Patches** is a repository containing personal, non-upstreamed patches for [ReVanced](https://revanced.app). These patches are maintained separately from the official ReVanced patches and offer additional functionality or modifications that haven't been merged into the main project.

## Overview

ReVanced is a modular patching framework for Android apps. This repository serves as a source for specific patches that you can apply to your applications using the ReVanced Manager or CLI.

## Usage

To use these patches, you typically need to include this repository as a patch source in your ReVanced patching tool.

### Using ReVanced CLI

If you are building patches using the CLI, you can specify the bundle depending on how you are building.

<Callout title="Note">
  Specific instructions for adding third-party patch sources may vary depending
  on the version of ReVanced Manager or CLI you are using.
</Callout>
